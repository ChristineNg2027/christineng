Viva

Web App React Firebase

Viva is a web platform for creating and sharing beautiful digital event invitations. The idea was simple: sending an invitation should feel as special as the event itself.

Users can design custom invitations, share them with a link, and collect RSVPs — all without needing to print a thing. It was a fun project to build end-to-end, from designing the UI to handling real-time data with Firebase.
